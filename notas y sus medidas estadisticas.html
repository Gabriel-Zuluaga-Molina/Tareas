## introducción

En este documento se están evaluando las notas de tres grupos de una materia(mañana, tarde y noche), y se calcularán algunas de las medidas estadisticas.

las notas del grupo de la ***mañana*** son: 4.0, 3.5, 3.8, 3.6, 4.1

las notas del grupo de la ***tarde*** son: 3.8, 3.5, 3.0, 2.9

las notas del grupo de la ***noche*** son: 3.1, 2.5, 2.6

```{r}
library(moments)
library(EnvStats)

# creamos vectores

mañana = c(4.0, 3.5, 3.8, 3.6, 4.1 )
tarde = c(3.8, 3.5, 3.0, 2.9)
noche = c(3.1, 2.5, 2.6)
todos = c(4.0, 3.5, 3.8, 3.6, 4.1 ,3.8, 3.5, 3.0, 2.9, 3.1, 2.5, 2.6)

cat("Datos cargados mañana:", mañana, "\n")
cat("Datos cargados tarde:", tarde, "\n")
cat("Datos cargados noche:", noche, "\n")

```

## Varianza de las medias(Intervarianza)

```{r}
medias = c(
  mean(mañana),
  mean(tarde),
  mean(noche)
)
# Tamaños de cada grupo
longitud = c(
  length(mañana),
  length(tarde),
  length(noche)
)

# numero de grupos

k = length(todos)

# Suma de cuadrados entre grupos (SS_between)
between = sum(longitud * (medias - mediato)^2)

intervarianza = between / k
cat("intervarianza:", intervarianza, "\n")
```

## Media de las varianzas(intravarianza)

```{r}
varianzas = c(
  varpoblacionalma,
  varpoblacionalta,
  varpoblacionalno
)

within = sum(varianzas * longitud)

intravarianza = within / k

cat("intravarianza:", intravarianza, "\n")
```
